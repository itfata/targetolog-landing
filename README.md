# Targetolog Mini Landing

Мини-лендинг с кастомным YouTube-плеером и отправкой заявок в Telegram.

## Запуск на Render

1. Загрузить проект в GitHub.
2. В Render создать новый Web Service из этого репозитория.
3. Build command: `npm install`
4. Start command: `npm start`
5. Добавить Environment Variables:
   - `TELEGRAM_BOT_TOKEN` - токен бота из BotFather
   - `TELEGRAM_CHAT_ID` - ID чата, куда должны приходить заявки

После этого форма отправляет заявки на `/api/lead`, а сервер пересылает их в Telegram.

## Проверка формы

Обычная страница:

```text
/
```

Страница с сразу открытой формой:

```text
/?form=1
```

## Проверка с локальным видео

Если есть файл видео в формате MP4, положите его сюда:

```text
outputs/video.mp4
```

После этого откройте страницу так:

```text
/?local=1
```

Или сразу с открытой формой:

```text
/?local=1&form=1
```

В локальном режиме работают те же ограничения плеера: назад перематывать можно, вперед дальше просмотренного места нельзя.

Чтобы использовать MP4 как основной вариант на финальном лендинге, поменяйте в `outputs/script.js`:

```js
videoMode: "youtube"
```

на:

```js
videoMode: "local"
```

## Настройки

Таймкод появления кнопки анкеты находится в `outputs/script.js`:

```js
ctaTimeSeconds: 180
```

Значение `180` означает, что кнопка появляется на 3-й минуте видео.
